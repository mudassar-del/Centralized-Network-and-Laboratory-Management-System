# Centralized Network and Laboratory Management System (CNLMS)

## Overview

The **Centralized Network and Laboratory Management System (CNLMS)** is a desktop-based application developed using **Python** and **PyQt6**. The system is designed to simplify the management of computer laboratories by providing centralized monitoring, communication, and administrative control.

It enables administrators to manage multiple client systems efficiently through a single interface, improving productivity and reducing manual effort.

---

## Features

* Centralized client monitoring
* Remote communication between server and clients
* Laboratory management
* Real-time status updates
* User-friendly graphical interface (PyQt6)
* Secure client-server communication
* Automated management operations

## Installation

1. Clone the repository

```bash
git clone https://github.com/mudassar-del/Centralized-Network-and-Laboratory-Management-System.git
```

2. Navigate to the project folder

```bash
cd Centralized-Network-and-Laboratory-Management-System
```

3. Install required dependencies

```bash
pip install -r requirements.txt
```

4. Run the application.
